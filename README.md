# MonsterHunter
C#物件導向專題

**魔物獵人-崛起**    
以ASP.NET創造出一個方便使用者查詢的系統，好讓使用者在遊戲進行中也能執行查詢的動作，以減少打錯魔物所浪費的時間。
同時使用者也能使用CRUD功能編輯資料庫。
